## üß† Key Frameworks

### The Duke Medical Taxonomy of AI Use
We do not view AI as binary (Allowed/Banned). We use a **Tiered Cognitive Framework**:

| Tier | Definition | The Rule |
| :--- | :--- | :--- |
| **Tier 0** | **Unaided Cognition** (Tech-Free) | *"Foundational memory must be protected."* |
| **Tier 1** | **AI-Restricted** (Search/Support) | *"AI is the Librarian, not the Author."* |
| **Tier 2** | **AI-Documented** (Drafting) | *"You are not the Writer; you are the **Sponsor**."* |
| **Tier 3** | **AI-Integrated** (Mandatory) | *"The human is the safety catch."* |

### Infrastructure as Access (The FCAP Pilot)
We are piloting a **Federated "Compute-to-Data" Bridge** to connect regional institutions (NCCU, Durham Tech) with real-world clinical datasets.
* **The Goal:** Democratize "Data Fluency" without compromising patient privacy.
* **The Pilot:** Launching Spring 2026.
